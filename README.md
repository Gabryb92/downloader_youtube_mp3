# Script per scaricare video da youtube e convertirli in mp3

### Creare un ambiente virtuale e attivarlo:

```python
python3 -m venv env
```

```bash
source env/bin/activate
```

### Installare i requisiti:

```python
python3 -m pip install -r requirements
```

### Lanciare il main.py:

```python
python3 main.py
```
